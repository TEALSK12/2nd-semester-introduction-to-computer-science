**TEALS Program** Home | Curriculum Map | Additional Readings | Change Log # IDE Recommendations ## Step 1: Install Python Before installing your IDE, first you will need to install Python. You can find the installation file and instructions here https://www.python.org/downloads/ ## Step 2: Decide and Install IDE Here is a list of common IDE’s (Integrated Development Environment) ### Visual Studio Code – (Open Source) Visual Studio Code is a free source-code editor made by Microsoft for Windows, Linux and macOS. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. Configuring VScode - https://youtu.be/EU8eayHWoZg ### Repl.it Repl.it is an online IDE that allows users to write code and build apps and websites using a browser, without having to install any software on their devices. It also allows teachers to create a classroom to post assignments along with having a build in grader. [www.repl.it](www.repl.it) ### Trinket.io Trinket lets you run and write code in any browser, on any device. Trinkets work instantly, with no need to log in, download plugins, or install software. Easily share or embed the code with your changes when you're done. Trinket has block based code that will let you see what it looks like in Python, along with the ability to create classroom to post assignments. [www.trinket.io](http://www.trinket.io) ### Pycharm PyCharm (Free for educators and students) Has auto complete and allows for import of libraries and easy project navigation. It is designed by programmers to provide all the tools needed to productive. [https://www.jetbrains.com/pycharm/](https://www.jetbrains.com/pycharm/) Install Guide - [https://www.jetbrains.com/help/pycharm/installation-guide.html](https://www.jetbrains.com/help/pycharm/installation-guide.html) ### Thonny Thonny (Free) It is designed for beginners and extremely easy to use. It comes with Python already installed so you do not have to install Python on to your computer first. Has code completion and it is not a big program so it does not take up a bunch of space. [https://thonny.org/](https://thonny.org/)
